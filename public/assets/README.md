# Project Rebuild Comprehensive Asset Pack

This folder is organized for direct use in the React/Vite PWA.

## Install

Copy the entire `assets` folder into:

```text
public/assets/
```

Then reference files with relative paths such as:

```js
image: './assets/exercises/push/machine-chest-press.jpg'
```

## Contents

- `exercises/` — independent strength-training visuals organized by movement family
- `stretches/` — independent mobility and flexibility visuals organized by body area
- `warmups/` — arrival warm-up and movement-preparation tiles
- `cardio/` — treadmill, bike, elliptical, and recovery-cardio tiles
- `recovery/` — breathing, sleep, hydration, and pain-check assets
- `ui/icons/` — navigation and metric icons
- `ui/badges/` — phase and achievement badges
- `ui/patterns/` — subtle Afrocentric-inspired geometric texture
- `ui/backgrounds/` — app hero and section backgrounds
- `data/asset-manifest.json` — complete file inventory
- `data/movement-catalog.json` — app-ready asset references and alt text

All movement visuals are independent files. None are cropped PDF pages or long infographic strips.
